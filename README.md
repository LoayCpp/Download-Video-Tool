# 🎥 Video Downloader App (C#)

A simple and lightweight C# Windows Forms application that allows users to download videos easily by pasting a video URL.

---

## 🚀 Features
- Simple and clean UI
- Paste video URL and download instantly
- Fast and easy to use
- Lightweight application

---

## 📌 How to Use

1. Open the application
2. Copy any video link (YouTube or supported source)
3. Paste the link into the textbox
4. Click the **Download** button
5. Wait for the download to complete

That’s it! 🎉 The video will be saved to your selected folder.
